Programa em C++ para auxiliar no aprendizado de Git.

Bem vindo(a) ao tutorial sobre Git.

Para a elaboração desse tutorial a nossa equipe optou por desenvolver um pequeno programa em C++ que auxiliará na aprendizagem da plataforma. Portanto dois requisitos são necessários para que possamos prosseguir.

  • Um compilador de C++ (Compatível com o standard de 2011)
  • O binário do Git instalado no seu computador

A instalação do Git depende da sua plataforma, portanto:

Se estiver no Windows:

Vá ao site: https://git-scm.com/download/win e faça o download da versão adequada ao seu windows.

Instale como se fosse programa normal do Windows

Se estiver usando Linux:

Se sua distribuição já suporta o pacote debian (Ubuntu, Mint e outros já suportam pacotes Debian por padrão, se estiver na dúvida, tente):


    sudo apt install git-all

Se não, você pode tentar:


    sudo yum install git

Após instalado, você pode conferir a instalação digitando git --version no console ou terminal.

Após instalar o Git:

Nós podemos usar git clone para clonar nosso programa que simula o workflow de uso de git. Crie um novo diretório, abra o terminal e digite:


    git clone https://github.com/spyci/programagitpraaprendergit

Após isso é só compilar.
