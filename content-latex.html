Primeiros passos:

1. Instalação:

Primeiramente, temos que fazer o download e a instalação do Latex no seu computador. Como é uma ferramenta multiplataforma, ou seja, funciona em diversos Sistemas Operacionais, vamos ensinar como fazer a instalação nos dois principais: Windows e Linux.

  • Para fazer a instalação no Windows, primeiro, faz-se a instalação do sistema tex, tendo como Link https://miktex.org/download. Isso também vale para o Linux, clicando na seção Linux.
  • Para Linux, o processo também é indicado no mesmo site para o Windows, porém, com os comandos para serem digitados no terminal. Atenção, pois deve-se levar em conta a distribuição Linux que se utiliza no PC.
  • Além disso, deve-se também fazer a instalação do editor para documentos Latex. Neste caso, ensinaremos fazer a instalação do TeXstudio. Para isso, basta clicar no site https://www.texstudio.org/ , que contém os Downloads tanto para Windows como para Linux. Para isso, basta clicar na seção "Downloads". Assim, só escolher o Sistema Operacional e a versão que se deseja instalar a ferramenta.
  • Após a instalação dos mesmos, basta seguir os próximos capítulos deste tutorial e ficar fera no uso de LaTeX.

Existe também uma plataforma TeX online chamada Overleaf que pode ser uma alternativa caso não esteja usando seu próprio computador. Neste caso, basta apenas entrar no site https://pt.overleaf.com/ , se registrar e começar o seu documento.

  • Para usar a plataforma, basta se registrar com um e-mail e uma senha da sua preferência e assim estarão disponíveis algumas ferramentas para começar ou dar continuidade aos seus projetos.
  • Assim que entrar, ao lado esquerdo da tela, procure pelo botão "Novo Projeto", e lá estarão algumas opções para escolher de acordo com sua preferência, além de vários modelos que poderá escolher de acordo com o que se precisa.
  • Esta plataforma também dispõe de outras ferramentas para organizar seus projetos e também versões pagas para melhorar a experiência do usuário. Possui também sincronização com o github para importar os projetos.

1. O Latex é uma ferramenta de compilação textual muito utilizada para a criação de documentos científicos. Apesar de ser um pouco complicado usar Latex no começo, todo o esforço vale a pena, pois os textos ficarão cada vez mais superiores aos feitos por editores de textos comuns.

2. A primeira coisa que se deve entender ao introduzir o Latex no cotidiano é a similaridade com códigos de programação, com isso, o mais comum a se aprender daqui pra frente são as funções para dar corpo ao texto e a utilidade das mesmas.

3. A primeira função a ser aprendida é: \documentclass{"tipo de documento"}

Esta, tem a finalidade de informar qual é o tipo de documento a ser produzido pelo autor. O "tipo de documento" pode ser trocado dependendo do documento que se deseja editar, como por exemplo "article", "report"(os nomes devem ser digitados em Inglês), entre outros. Dependendo do que se precisa, é uma função importante para informar o leitor o tipo do documento que está sendo lido.

4. Segundo, e não menos importante, temos as funções de começo e fim de documento, que servem (como o próprio nome já diz) para marcar qual é o início e qual é o fim do documento:

    \begin{document}

            O corpo do seu documento deve ser escrito entre essas duas funções.

    \end{document}

5. Quanto à formatação básica:

  1. Todos os comandos Latex começam com o símbolo \ (barra invertida);
  2. Os parágrafos são do tipo Justificado, ou seja, alinhados à esquerda e a direita;
  3. O espaçamento é automático;
  4. Para iniciar um novo parágrafo deixa-se uma ou mais linhas em branco. A primeira linha possui recuo (geralmente);
  5. As linhas iniciadas com o símbolo % não serão consideradas pelo programa, servindo de comentário para anotações;
  6. Para fazer uma quebra de página adiciona-se o comando \newpage;
  7. Para fazer uma quebra de linha adiciona-se o comando \newline;
  8. A indentação pode ser evitada usando o comando \noindent;
  9. Para centralizar linhas usa-se o comando \centerline;
  10. Para linhas mais à esquerda e mais à direita usa-se respectivamente \leftline e \rightline.
  11. Os caracteres # $ % & ~ _ ^ \ { } não podem ser utilizados normalmente, pois são reservados para designar algumas funções, assim, existem outras maneiras de representá-los. Para que algum desses apareça no texto, basta adicionar \ na frente.
